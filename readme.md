# App with Authentication using AWS Amplify

### Sign in screen

- Username input
- Password input
- Buttons with an option to sign in using: Facebook, Google, Apple.

### Sign up screen

- Username input
- Password input
- Email input
- Buttons with an option to sign in using: Facebook, Google, Apple.

### Confirm email screen

- code input that is sent to email
- Back to sign in button
- Resend code button
